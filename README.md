# Data Simulations — Student Projects

## Mục tiêu
Dự án trình bày các phân tích dữ liệu và mô phỏng thống kê dùng Pandas, NumPy và trực quan hóa bằng Matplotlib.

## Nội dung chính
- `notebooks/`: Jupyter notebooks với phân tích dữ liệu và mô phỏng Monte Carlo.
- `data/`: bộ dữ liệu dùng cho ví dụ.
- `src/`: script thực thi nhanh (Python).
- `results/figures/`: biểu đồ xuất ra từ các phân tích.

## Cách chạy (local)
1. Cài environment (tùy chọn):
   - `pip install -r requirements.txt`
2. Chạy notebook bằng Jupyter:
   - `jupyter notebook notebooks/monte_carlo_simulation.ipynb`
3. Hoặc chạy script:
   - `python src/analysis.py`

## Kết quả
- Biểu đồ histogram (saved at `results/figures/montecarlo_plot.png`) minh họa phân bố mean từ các mô phỏng.
- Trong báo cáo, chèn các ảnh trong `results/figures/`.

## Nguồn dữ liệu
- Nếu dùng dataset từ Kaggle/UCI, ghi rõ link và ngày tải.
